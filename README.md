# web-project
ICT280 - Intro to Web Development - Project 1

Project 2 - HTML Foundation Page:

1.) The theme of my webpage is to give some information about Las Cruces, NM and some interesting facts about it.
I included a home page, an activities page, and a page about the weather. I may consider to add a few more pages in the next part 
of the project like an NMSU section, demographics, or cultural/history.

2.) I learned how to create a web page with multiple navigational elements, how to embed pictures and links, and how to use HTML
styling elements to alter the look and feel of the page. I learned how to use https://html.onlineviewer.net/ to see a live preview
of my page so I could see how the HTML affected the website.

3.) I thought that only using HTML was challenging whenever I needed to use the Nu HTML Checker to verify my code. I kept getting
several warnings and errors that informed me that my code was obsolete and to use CSS instead. I wasn't sure whether I needed to remove
the HTML style components I used or not, even though they worked properly on the HTML online viewer.
