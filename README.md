# web-project
ICT280 - Intro to Web Development - Project 1

Project 2 - HTML Foundation Page:

1.) The theme of my webpage is to give some information about Las Cruces, NM and some interesting facts about it.
I included a home page, an activities page, and a page about the weather. I may consider to add a few more pages in the next part 
of the project like an NMSU section, demographics, or cultural/history.

2.) I learned how to create a web page with multiple navigational elements, how to embed pictures and links, and how to use HTML
styling elements to alter the look and feel of the page. I learned how to use https://html.onlineviewer.net/ to see a live preview
of my page so I could see how the HTML affected the website.

3.) I thought that only using HTML was challenging whenever I needed to use the Nu HTML Checker to verify my code. I kept getting
several warnings and errors that informed me that my code was obsolete and to use CSS instead. I wasn't sure whether I needed to remove
the HTML style components I used or not, even though they worked properly on the HTML online viewer.

Project 3 - Addition of CSS to Webpage:

1.) I kept the theme of my website the same and retained all of the same text, images, links and other aspects of the page. I decided to make some adjustments to my index.html page to include working links to both my Project 2 and Project 3 so that they could be compared, from the index page you can choose to look at Project 2 (with all the same functionality and links in the navigation bar) or Project 3 (with the new implementation of a CSS stylesheet and unncessary HTML elements removed).

2.) This time I learned how to create a CSS stylesheet to apply rules across all pages in my website. I also learned how to make an Images directory in my repo and how to embed images directly from my files as opposed to using URLs from online. I figured out how to apply a container class which kept my formatting for things like margin and text-format uniform and added a border to make everything look neat and presentable. 

3.) Surprisingly I didn't run into as many issues in this section of our project, all I needed to do was create my CSS stylesheet and remove obsolete HTML elements that I had used as a placeholder for CSS in the last project. Updating my repository went very smoothly, and I didn't need to spend a lot of time fixing broken links because I had spent so much time figuring it out last time. I used https://www.codechef.com/html-online-compiler to help me visualize how my index.html and styles.css would interact with each other in real-time.
